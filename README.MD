A modern, feature-rich Client Lead Management System (Mini CRM) built with pure HTML, CSS, and JavaScript. No backend required - everything runs in your browser with local storage persistence.

![LeadManager Pro Dashboard](https://via.placeholder.com/800x400?text=LeadManager+Pro+Dashboard)

##  Features

### Core Functionality
- **Lead Management** - Complete CRUD operations for client leads
- **Status Tracking** - Track leads through workflow stages (New → Contacted → Converted)
- **Notes System** - Add and manage follow-up notes for each lead
- **Search & Filter** - Find leads by name, email, or status
- **Analytics Dashboard** - Visual insights with interactive charts

### User Interface
- **Modern Design** - Gradient backgrounds, smooth animations, and glassmorphism effects
- **Responsive Layout** - Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements** - Hover effects, modal dialogs, and toast notifications
- **Dark/Light Theme** - Professional color scheme with high contrast

### Data Management
- **Local Storage** - Automatic data persistence in browser
- **Sample Data** - Pre-loaded demo leads for testing
- **Real-time Updates** - Instant UI updates on data changes
- **Data Export Ready** - Easy to extend for CSV/JSON export
